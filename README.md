# Darter Morph

R scripts for the darter morphometrics pipeline.

## Contents

- `R/00_GPA.R` - core morphometric initialization
- `R/01_metadata.R` - specimen metadata aligned to GPA coordinates
- `R/01_helpers_angles.R` - helper functions for angle measurements
- `R/01_build_angle_measurements.R` - angle measurement construction
- `R/02_subset_1950.R` - 1950 specimen subset
- `R/03_subset_CT_timeseries.R` - CT time-series subset
- `R/04_subset_CT_timeseries_plus_1950habitats.R` - CT time-series plus 1950 habitats subset

## Notes

The repository currently includes the R scripts from `Documents/darter_pipeline/R`.
Input data, photos, generated figures, and analysis outputs are not included.
